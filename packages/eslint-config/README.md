Shared ESLint config.

## Install

```sh
# npm
npm install --save-dev @shooontan/eslint-config eslint prettier

# or yarn
yarn add --dev @shooontan/eslint-config eslint prettier
```

## Usage

```js
module.exports = {
  extends: ['@shooontan'],
};
```
