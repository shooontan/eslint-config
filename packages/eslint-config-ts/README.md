Shared ESLint config for TypeScript.

## Install

```sh
# npm
npm install --save-dev @shooontan/eslint-config-ts eslint prettier

# or yarn
yarn add --dev @shooontan/eslint-config-ts eslint prettier
```

## Usage

```js
module.exports = {
  extends: ['@shooontan/eslint-config-ts'],
};
```
